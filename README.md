# React PHP CRUD

- Install node, npm 
- Create React app: npx create-react-app react-php-crud
- Create Links and routing in React: npm i react-router-dom V6
```jsx
import { BrowserRouter, Routes, Route, Link } from 'react-router-dom';
```
